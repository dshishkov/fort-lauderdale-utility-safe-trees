# Fort Lauderdale Utility-Safe Trees

This repository contains a polished static reference site for the City of Fort Lauderdale's `15'-20'` small shade trees that may be planted below neighborhood overhead power lines.

## Contents

- `index.html` - the full static website
- `citytrees.pdf` - source city tree list referenced while rebuilding the guide

## Notes

- The page focuses only on the city's under-wire `15'-20'` category.
- Each tree entry includes multiple example photos and a short note about likely root risk near foundations and underground utilities.
- Remote images are loaded from open photo sources.
